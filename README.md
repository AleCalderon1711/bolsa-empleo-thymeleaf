# Bolsa de Empleo v1

Job board application built with Spring Boot, Thymeleaf and MySQL.

## Description

Web platform that connects job seekers with companies. Built with server-side rendering using Thymeleaf. Users can register as a company or job seeker, manage job listings, apply to positions, and track applications. Administrators manage platform access and approvals.

## Features

- Role-based access control (Admin, Company, Job Seeker)
- Spring Security authentication
- Company dashboard to publish and manage job listings
- Job seeker dashboard with CV, skills and applications
- Admin panel to approve companies and job seekers
- Server-side rendering with Thymeleaf fragments

## Tech Stack

- **Frontend:** Thymeleaf, HTML, CSS, Bootstrap
- **Backend:** Java, Spring Boot, Spring Security, JPA
- **Database:** MySQL
